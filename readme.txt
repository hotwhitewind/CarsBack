Для запуска необходимо создать БД с помощью migration.bat из cmd или вызова команды Update-Database
в среде VisualStudio
Если неустановлены ef tools, нужно установить с помощью install_ef_tools.bat
Строку подключения к БД задать в appsettings.json
Тестирование можно провести через Swagger. Страница автоматически
загрузится при запуске проекта в IISExpress
